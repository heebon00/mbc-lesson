😎 CSS 이름표(ID) vs 반티(Class) 한방 정리

1. ID (아이디) - "#" 기호 사용

비유: 내 이름표 / 주민등록번호

  • 우리 학교에 내 이름이랑 똑같은 애가 한 명 더 있으면 헷갈리겠지?
  • 그래서 ID는 HTML 문서 안에서 딱 한 군데만 쓸 수 있어!
  • CSS에서는 #을 이름 앞에 붙여서 꾸며줘.

나만 골드 이름표야! (id="super-hero")

2. Class (클래스) - "." 기호 사용

비유: 우리 반 반티 / 축구팀 유니폼

  • 우리 반 친구들 다 같이 똑같은 옷 입을 수 있지?
  • Class는 여러 명(태그)이 한꺼번에 똑같은 스타일을 입고 싶을 때 써!
  • CSS에서는 .을 이름 앞에 붙여서 꾸며줘.

나는 파란팀! (class="blue-team")

나도 파란팀! (class="blue-team")

우리 다 같이 파란색! (class="blue-team")

3. 잼민이를 위한 핵심 요약!

구분 ID (아이디) Class (클래스)
기호 # (샵) . (점)
개수 한 번만 가능 (유니크) 여러 번 가능 (중복)
언제 써? 진짜 특별한 놈 하나만 꾸밀 때 여러 놈을 한꺼번에 꾸밀 때

이제 안 헷갈리겠지? 열공해! 🔥
