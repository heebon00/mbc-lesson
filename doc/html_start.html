🏠잼민이의 HTML 기본 구조 공부 노트

HTML은 웹사이트를 만드는 설계도예요!

1. HTML의 기본 뼈대 (코드)

<!DOCTYPE html>
<html>
  <head>
    <title>내 첫 웹사이트</title>
  </head>
  <body>
    <h1>안녕, 반가워! 난, 잼민이야.</h1>
    <p>여기에 내용을 적어요.</p>
  </body>
</html>

2. 각 부분이 하는 일

<!DOCTYPE html> : "이건 최신 HTML 문서야!"라고 알려주는 시작 인사예요.
<html> : 집의 전체 울타리예요. 모든 내용은 이 안에 들어가야 해요.
<head> : 웹사이트의 정보(제목, 설정 등)를 담는 '두뇌' 역할이에요. 눈에는 안 보여요!
<body> : 우리 눈에 보이는 진짜 내용(글, 그림, 버튼)이 사는 곳이에요.

3. 잼민이 코딩 꿀팁!

  • 열고 닫기: 태그는 시작했으면 반드시 </태그>로 닫아줘야 해요. (뚜껑 닫기!)
  • 들여쓰기: Tab 키를 눌러서 안으로 밀어 쓰면 코드가 훨씬 예뻐 보여요.
