Gemini CLI Slash Commands Summary

Gemini CLI에서 사용할 수 있는 주요 슬래시 명령어(/) 목록입니다. 이 명령어들을 통해 세션 관리, 설정 변경 및 다양한 메타 기능을 제어할 수 있습니다.

명령어 설명
/aboutCLI 버전 정보를 표시합니다.
/agents로컬 및 원격 서브 에이전트를 관리합니다 (목록, 리로드, 활성화/비활성화, 설정).
/auth인증 방법을 변경하는 대화 상자를 엽니다.
/bugGitHub에 Gemini CLI 관련 이슈를 제보합니다.
/chat (또는 /resume)이전 세션을 찾아보거나 재개하고, 수동 체크포인트를 관리합니다.
/clear터미널 화면과 스크롤백을 지웁니다 (단축키: Ctrl+L).
/commands.toml 파일에서 로드된 사용자 정의 슬래시 명령어를 관리합니다.
/compress현재 대화 컨텍스트를 요약으로 대체하여 토큰을 절약합니다.
/copy마지막 Gemini CLI 출력을 시스템 클립보드에 복사합니다.
/directory (또는 /dir)다중 디렉토리 지원을 위한 워크스페이스 디렉토리를 관리합니다.
/docs기본 웹 브라우저에서 Gemini CLI 문서를 엽니다.
/editor지원되는 에디터를 선택하는 대화 상자를 엽니다.
/extensions확장 기능(extensions)을 관리합니다 (설치, 제거, 목록, 활성화/비활성화 등).
/help (또는 /?)사용 가능한 명령어와 사용법에 대한 도움말 정보를 표시합니다.
/hooksCLI 동작을 가로채고 커스터마이징하기 위한 라이프사이클 훅을 관리합니다.
/ideIDE 통합 설정 및 상태를 관리합니다.
/init디렉토리를 분석하고 맞춤형 GEMINI.md 컨텍스트 파일을 생성합니다.
/mcpModel Context Protocol (MCP) 서버 및 도구를 관리합니다.
/memoryGEMINI.md 파일에서 로드된 계층적 지침 메모리를 관리합니다.
/model모델 설정을 관리합니다 (모델 전환 또는 설정 대화 상자 열기).
/permissions폴더 신뢰 설정 및 기타 권한을 관리합니다.
/plan계획 모드(읽기 전용)로 전환하고 현재 실행 계획을 확인합니다.
/policies활성화된 모든 보안 및 도구 사용 정책을 나열합니다.
/privacy개인정보 보호 고지를 표시하고 데이터 수집 동의를 관리합니다.
/quit (또는 /exit)Gemini CLI 세션을 종료합니다 (선택적으로 세션 데이터 삭제 가능).
/restore도구 실행 전의 상태로 프로젝트 파일을 복원합니다.
/rewind대화 기록을 뒤로 이동하여 대화나 코드 변경 사항을 되돌립니다.
/settingsGemini CLI 설정을 수정하는 대화형 에디터를 엽니다.
/shells (또는 /bashes)백그라운드 쉘 프로세스 뷰를 토글합니다.
/setup-github이슈 분류 및 PR 리뷰를 위한 GitHub Actions를 설정합니다.
/skills에이전트 기술(Agent Skills)을 관리합니다 (활성화, 비활성화, 목록, 리로드).
/stats세션, 모델 또는 도구 사용 통계를 표시합니다.
/terminal-setup여러 줄 입력을 위한 터미널 키바인딩을 설정합니다.
/themeCLI의 시각적 테마를 변경하는 대화 상자를 엽니다.
/tools사용 가능한 도구와 그 설명을 나열합니다.
/upgrade브라우저에서 Gemini Code Assist 업그레이드 페이지를 엽니다.
/vimVim 스타일의 탐색 및 편집 모드를 켜거나 끕니다.
