🪄 CSS 선택자 마법 학교

안녕 잼민아! CSS 선택자는 "누구를 꾸밀지 찍어주는 화살표"라고 생각하면 돼. 화살표를 잘못 쏘면 엉뚱한 애가 꾸며지겠지? 하나씩 배워보자!

1. 전체 선택자 (*) - "야! 다 모여!"

문서에 있는 모든 태그를 한꺼번에 꾸밀 때 써.

* { color: gray; }

👉 학교 운동장에 있는 모든 사람한테 "회색 옷 입어!"라고 외치는 거야.

2. 태그 선택자 (이름) - "000 들어간 애들 다 나와!"

특정 태그 이름(p, h1, div 등)을 가진 애들을 몽땅 꾸며.

p { font-size: 15px; }

👉 "우리 학교에 있는 모든 p학생들은 키 15만큼 커져라!"

3. 클래스 선택자 (.) - "반티 입은 사람들!"

태그에 class="이름"이라고 적힌 애들을 꾸며. 앞에 점(.)을 찍는 거 잊지 마!

.blue-team { color: blue; }

나 블루팀! 난 그냥 사람. 나도 블루팀!

4. 아이디 선택자 (#) - "너 딱 한 명만 이리와!"

태그에 id="이름"이라고 딱 하나만 적힌 애를 꾸며. 앞에 샵(#)을 붙여!

#king { color: purple; }

나는 이 구역의 킹이다 (id="king")

5. 자손 선택자 (한 칸 띄우기) - "~안에 있는 ~"

어떤 태그 안에 들어있는 다른 태그만 콕 집어.

div p { color: green; }

👉 "교실(div) 안에 있는 학생(p)들만 초록색으로 변해라!" (교실 밖에 있는 학생은 안 변함)

6. 호버 선택자 (:hover) - "마우스가 닿으면 변신!"

마우스를 올렸을 때만 스타일이 뿅 나타나.

button:hover { background: pink; }

💡 마지막 꿀팁!

- #아이디: 세상에 단 하나뿐인 주인공 (주민번호)
- .클래스: 여러 명이 같이 입는 팀복 (유니폼)
- 태그: 그냥 그 종류 전체 (생물종)

이제 너도 CSS 마법사! 🧙‍♂️✨
