🎮 HTML 태그 쉽게 이해하기

블록 레벨(Block) vs 인라인 레벨(Inline) 완벽 정리!

🤔 먼저 알아두기

HTML 태그는 크게 두 가지 성격을 가지고 있어요!

마치 혼자 한 줄을 다 차지하는 왕과 옆에 붙어 다니는 친구 같은 느낌이에요 😄

💡 쉽게 외우는 법:
• 블록(Block) = "나 혼자 한 줄!" (떼놓은 닭처럼 혼자!)
• 인라인(Inline) = "옆에 붙어도 돼!" (친구랑 나란히!)

🧱 블록 레벨 태그 (Block Level)

특징: 무조건 한 줄을 다 차지해요! 옆에 아무것도 못 붙어요!

마치 혼자서 한 줄을 다 쓰는 주인공 같은 존재예요 🌟

👇 실제로 보면 이렇게 나와요:

<div> 태그
<p> 태그
<h1> 태그

📋 주요 블록 태그들

<div> <p> <h1>~<h6> <ul> <ol> <li> <header> <footer> <section> <article> <nav> <aside> <main> <form> <table> <blockquote> <pre> <hr>
<!-- 블록 태그 예시 --> <div>나는 div! 한 줄 다 차지해!</div> <p>나는 문단! 나도 한 줄!</p> <h1>큰 제목! 나도 혼자!</h1>

🔗 인라인 레벨 태그 (Inline Level)

특징: 필요한 만큼만 공간을 쓰고, 옆에 다른 태그가 붙을 수 있어요!

마치 손잡고 옆으로 나란히 서는 친구들 같아요 👫

👇 실제로 보면 이렇게 나와요:

<span> <a> <strong> <em> <img>

📋 주요 인라인 태그들

<span> <a> <strong> <em> <img> <input> <label> <button> <textarea> <select> <br> <code> <mark> <small> <sub> <sup> <b> <i> <u>
<!-- 인라인 태그 예시 --> <span>나는 span</span> <a href="#">링크도 옆에!</a> <strong>굵게도 옆에!</strong>

⚖️ 한눈에 비교하기

특징 🧱 블록 레벨 🔗 인라인 레벨
줄 바꿈 무조건 다음 줄로! 옆에 계속 붙음!
너비 부모의 100% (한 줄 다!) 내용물 크기만큼만!
높이 설정 가능 (height) 기본적으로 설정 어려움
margin/padding 상하좌우 다 됨! 좌우만 기본 적용!
안에 뭘 넣을 수 있나 블록 + 인라인 다 가능! 보통 인라인만! (블록 넣으면 깨짐)

🛠️ 실전 예제

아래 코드를 보면서 차이를 느껴봐요!

<!-- 블록 안에 인라인 넣기 (OK!) --> <div> 나는 div(블록) 안에 <span>span(인라인)</span>을 넣었어! </div> <!-- 인라인 안에 블록 넣기 (NO!) --> <!-- 이렇게 하면 브라우저가 이상하게 표시할 수 있어요 --> <span> <div>이렇게 하면 안 돼요!</div> </span>
🎯 핵심 정리:
• 블록 태그는 집(컨테이너) 같은 역할 → 안에 뭐든 넣을 수 있음
• 인라인 태그는 꾸미는 도구 같은 역할 → 글자를 예쁘게 만듦
• div는 만능 블록 상자, span은 만능 인라인 꾸미개!

🎉 이제 HTML 태그의 블록/인라인 개념 마스터!
